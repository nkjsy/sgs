# 标准版官方牌堆对照清单（当前项目）

更新时间：2026-02-22

## 结论

当前项目已切换为“标准身份场 108 口径（标准 + EX 并入）+ 显式花色点数”生成方式。

- 当前牌堆定义于 `packages/core/src/cards.ts`。
- 当前阶段范围约束见 `docs/scope-standard-identity.md`。
- 规则能力仍是子集实现（尤其装备技能），见 `docs/rule-coverage.md`。

## 对照口径

为避免口径混乱，本清单采用以下口径：

1. 以官方规则卡牌页（基本牌/锦囊牌/装备牌）作为名称来源。
2. 本清单只核对“是否具备该牌种与基础结算能力”，不把武将技能联动算作已完整。
3. 当前代码以 `packages/core/src/cards.ts` 内显式蓝图为准，保持“可运行+可回归”的工程口径。

### 基本牌

- 已实现：杀、闪、桃（当前范围已覆盖）。

### 锦囊牌（常见标准/国标交集）

- 已实现：
  - 过河拆桥、顺手牵羊、决斗、借刀杀人、无中生有、无懈可击、万箭齐发、南蛮入侵、桃园结义、五谷丰登、乐不思蜀、闪电
- 规则页存在但当前不在本项目“标准身份场范围”内：
  - 铁索连环、火攻、兵粮寸断、以逸待劳、知己知彼、远交近攻等（多属于军争/国战扩展口径）

### 装备牌

- 牌名壳与装备槽位已补齐到标准身份场常见集合。
- 已实现的装备技能子集：诸葛连弩、八卦阵、仁王盾、青釭剑、青龙偃月刀、寒冰剑、贯石斧、麒麟弓、丈八蛇矛、方天画戟、雌雄双股剑（基础效果）。
- 仍缺失的是“装备技能完整规则”（复杂联动/时序细则尚未覆盖）。
- 不在当前主线范围（军争/国战等扩展）：
  - 古锭刀、朱雀羽扇、白银狮子、藤甲、吴六剑、三尖两刃刀等

## 关键缺口类型

1. **技能细节缺口**：已实现牌种下，装备技能与复杂结算仍有官方细则未完全覆盖。
2. **时序边界缺口**：无懈链、延时锦囊、双目标锦囊仍有边界未全量覆盖。
3. **武将系统缺口**：标准将池技能系统尚未接入。

## 最小待办（标准身份场主线）

1. 补装备技能“细时序”回归（不新增扩展牌）。
2. 补无懈链/延时锦囊复杂边界回归。
3. 落地标准将池技能框架（仍锁定身份场）。
