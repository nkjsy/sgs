# 标准版官方牌堆对照清单（当前项目）

更新时间：2026-02-22

## 结论

当前项目已切换为“标准身份场官方牌堆口径（108）+ 显式花色点数”生成方式。

- 当前牌堆定义于 `packages/core/src/cards.ts`。
- 当前阶段范围约束见 `docs/scope-standard-identity.md`。
- 规则能力仍是子集实现（尤其装备技能），见 `docs/rule-coverage.md`。

## 对照口径

为避免口径混乱，本清单采用以下口径：

1. 以官方规则卡牌页（基本牌/锦囊牌/装备牌）作为名称来源。
2. 本清单只核对“是否具备该牌种与基础结算能力”，不把武将技能联动算作已完整。
3. 官方“每张牌的精确数量、花色、点数分布”在当前引用页面中未直接给出完整表，因此统一标注为“待补官方明细表”。

## 差异清单（按牌种）

### 基本牌

- 已实现：杀、闪、桃
- 缺失：
  - 酒（规则页标注为军争/国标体系常见，当前未实现）

### 锦囊牌（常见标准/国标交集）

- 已实现：
  - 过河拆桥、顺手牵羊、决斗、借刀杀人、无中生有、无懈可击、万箭齐发、南蛮入侵、桃园结义、五谷丰登、乐不思蜀、闪电
- 规则页存在但当前不在本项目“标准身份场范围”内：
  - 铁索连环、火攻、兵粮寸断、以逸待劳、知己知彼、远交近攻等（多属于军争/国战扩展口径）

### 装备牌

- 牌名壳与装备槽位已补齐到标准身份场常见集合。
- 已实现的装备技能子集：诸葛连弩、八卦阵、仁王盾、青釭剑（基础效果）。
- 仍缺失的是“装备技能完整规则”（其余武器/防具技能与复杂联动尚未覆盖）。
- 扩展口径未实现（EX/军争/国战等）：
  - 寒冰剑、仁王盾、朱雀羽扇、白银狮子、藤甲、吴六剑、三尖两刃刀等

## 关键缺口类型

1. **牌种缺口**：仍缺若干标准常用牌（至少包括无中生有、完整装备集）。
2. **技能缺口**：装备技能与复杂结算仍是 MVP 简化版。
3. **边界缺口**：无懈链、延时锦囊、双目标锦囊仍有官方细则未全量覆盖。

## 建议补齐顺序

1. 补装备技能（中高风险）：按牌名逐个补齐技能触发与时机。
2. 补官方边界（中高风险）：无懈链/借刀/延时锦囊细则。
3. 补武将系统（高风险）：仅标准将池，保持身份场口径。
