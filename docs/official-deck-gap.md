# 标准版官方牌堆对照清单（当前项目）

更新时间：2026-02-22

## 结论

当前项目**未完成**三国杀标准版官方牌堆的全量实现。

- 当前牌堆来自 MVP 自定义分布，定义于 `packages/core/src/cards.ts`。
- 规则能力也属于子集实现，见 `docs/rule-coverage.md`。
- 当前阶段范围约束见 `docs/scope-standard-identity.md`。

## 对照口径

为避免口径混乱，本清单采用以下口径：

1. 以官方规则卡牌页（基本牌/锦囊牌/装备牌）作为名称来源。
2. 本清单只核对“是否具备该牌种与基础结算能力”，不把武将技能联动算作已完整。
3. 官方“每张牌的精确数量、花色、点数分布”在当前引用页面中未直接给出完整表，因此统一标注为“待补官方明细表”。

## 差异清单（按牌种）

### 基本牌

- 已实现：杀、闪、桃
- 缺失：
  - 酒（规则页标注为军争/国标体系常见，当前未实现）

### 锦囊牌（常见标准/国标交集）

- 已实现：
  - 过河拆桥、顺手牵羊、决斗、借刀杀人、无中生有、无懈可击、万箭齐发、南蛮入侵、桃园结义、五谷丰登、乐不思蜀、闪电
- 规则页存在但当前不在本项目“标准版MVP”范围：
  - 铁索连环、火攻、兵粮寸断、以逸待劳、知己知彼、远交近攻等（多属于军争/国战扩展口径）

### 装备牌

- 当前仅实现抽象装备子集：
  - 1 种武器效果（攻击范围+1）
  - +1/-1坐骑
- 明确缺失（常见标准装备牌与技能）：
  - 诸葛连弩、青釭剑、雌雄双股剑、贯石斧、青龙偃月刀、丈八蛇矛、方天画戟、麒麟弓、八卦阵等
- 扩展口径未实现（EX/军争/国战等）：
  - 寒冰剑、仁王盾、朱雀羽扇、白银狮子、藤甲、吴六剑、三尖两刃刀等

## 关键缺口类型

1. **牌种缺口**：仍缺若干标准常用牌（至少包括无中生有、完整装备集）。
2. **数量缺口**：已实现牌种的张数并非官方标准牌堆定值。
3. **牌面信息缺口**：当前未落地官方花色/点数/属性杀分布。
4. **技能缺口**：装备技能与复杂结算仍是 MVP 简化版。

## 建议补齐顺序

1. 补牌种（低风险）：先加无中生有 + 标准武器/防具名称壳。
2. 补数量（中风险）：将 `createDeck()` 调整为官方牌堆数量分布。
3. 补牌面（中高风险）：加入花色、点数、属性杀，联动判定/技能。
4. 补技能（高风险）：装备技能与无懈/时序深度规则。

## 你当前问的“是否已完成官方标准牌堆”

答案：**没有完成**。

当前状态是“可运行的规则引擎 MVP + 部分牌种实现”，不是官方标准版全量牌堆。
